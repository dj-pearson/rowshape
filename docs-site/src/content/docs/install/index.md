rowshape is a single static Go binary. There is no runtime to install and no
services to run.

Install instructions land here with the first tagged release, covering Homebrew,
`go install`, the npm wrapper (`npx rowshape`), the `FROM scratch` Docker image
for CI, and direct downloads from GitHub Releases for macOS, Linux, and Windows on
both amd64 and arm64.

Every release ships an SBOM and a cosign signature.
