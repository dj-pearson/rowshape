The fixture format is an open spec with its own repository and its own version.
Anyone can emit it; rowshape is one implementation, not the definition.

RFC-0001 covers the document shape, the confidence model (every fact carries how
well it is known), the privacy levels, and the canonical form used to compute a
fixture's digest.

The spec lives at [rowshape/fixture-spec](https://github.com/rowshape/fixture-spec).
